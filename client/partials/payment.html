Simple implementation of Paying Bitcoin to a Ethereum Wallet Address That is stored on the server. The server uses the ShapeShift API to convert the Bitcoin to Ethereum. To pay the configured wallet input the amount of Ethereum you would like to send and optionally a return address for your Bitcoin in case something goes wrong with the transaction and your funds need to be returned. Click Start Payment which then should return a QR code of the address to Deposit your Bitcoin for the transaction as well the amount of Bitcoin you need to send in able to convert to the desired amount of Ethereum you want to send. If there is an error, such as setting the amount of Ethereum you want to send to high or to low an error will be displayed.
